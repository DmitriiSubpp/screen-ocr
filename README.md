## Установка

- 1
```bash
$ git clone https://github.com/dmitrii-subb/screen-ocr.git
$ cd screen-ocr
$ pip install -r requirements.txt
```
- 2
Распаковать архив Tesseract-OCR и поместить в папку с screen-ocr.py
